# Pet

Provides information about the player's pet.

## Table
- **index**
- **id**
- **name**
- **owner_index**
- **owner_id**
- **hp** (pup only)
- **hp_max** (pup only)
- **hp_percent**
- **mp** (pup only)
- **mp_max** (pup only)
- **mp_percent**
- **tp**
- **active**: true is pet is active (logic might need further refinement)
